About This Application


This website uses data available from the GitHub API. As such, only data for publicly viewable repositories can be collected.

Determination of a developer's location is done using the free-form data that they entered into their GitHub profile. For example, if the search is on "Victoria BC, Canada" but the developer didn't include the word Victoria (case insensitive) in their location, specifying for instance Saanich which is part of Victoria, then that developer's location will not match the query.
