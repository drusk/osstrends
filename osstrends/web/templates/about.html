About This Application


This website uses data available from the GitHub API. As such, only data for publicly viewable repositories can be collected.

Determination of a user's location is done using the free-form data that user's are able to enter into their GitHub profile. For example, if the search is on "Victoria BC, Canada" but the user doesn't include the word Victoria in their location (using for instance Saanich which is part of Victoria) then that user will not be recognized.
